## Objective

대시보드에서 **자율 레벨(Level 1/2/3)**을 선택하고,
실행 중인 **Plan의 진행 상태를 시각화**하는 UI를 구축한다.

```
Level 1 (Manual):     매 단계마다 사용자 승인
Level 2 (Semi-Auto):  계획 승인 후 실행은 자동, 에러 시 중단
Level 3 (Full-Auto):  계획 수립부터 실행·검토까지 자율 (위험 작업만 승인)
```

#### PlannerUseCase 수정 (~20줄)

자율 레벨에 따른 동작:
- **Level 1**: 모든 step에 requiresApproval = true (강제 적용)
- **Level 2**: 계획 생성 시 승인 필요, 이후 자동 실행. step 실패 시 paused
- **Level 3**: 계획 자동 실행. `delegate_task`와 `write_file`만 승인 필요

```typescript
constructor(
  // ... 기존
  private readonly settingsRepository: ISettingsRepository,
) {}

private shouldRequireApproval(step: PlanStep): boolean {
  const level = this.settingsRepository.getAutonomyLevel();
  if (level === 1) return true;
  if (level === 2) return step.requiresApproval; // 원래 설정 유지
  // level === 3: 위험 작업만
  return ["delegate_task", "stop_agent", "write_file"].includes(step.action);
}
```

### 2. 대시보드 — 자율 레벨 선택 UI

#### dashboard/components/autonomy-selector.tsx (신규, ~60줄)

Overview 탭 상단에 배치. 3개 버튼 그룹:

```tsx
export function AutonomySelector() {
  const [level, setLevel] = useState<1 | 2 | 3>(1);

  useEffect(() => {
    fetch("/api/settings/autonomy").then(r => r.json()).then(d => setLevel(d.level));
  }, []);

  const handleChange = async (newLevel: 1 | 2 | 3) => {
    await fetch("/api/settings/autonomy", {
      method: "PUT",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({ level: newLevel }),
    });
    setLevel(newLevel);
  };

  return (
    <div className="flex items-center gap-2 p-3 bg-card rounded-lg border">
      <span className="text-sm font-medium text-muted-foreground">Autonomy:</span>
      {([1, 2, 3] as const).map((l) => (
        <button
          key={l}
          onClick={() => handleChange(l)}
          className={cn(
            "px-3 py-1 text-xs rounded-md transition-colors",
            level === l
              ? "bg-primary text-primary-foreground"
              : "bg-muted text-muted-foreground hover:bg-muted/80"
          )}
        >
          L{l}
        </button>
      ))}
      <span className="text-xs text-muted-foreground ml-2">
        {DESCRIPTIONS[level]}
      </span>
    </div>
  );
}
```

### 3. 대시보드 — Plan 진행 시각화

#### dashboard/components/plan-progress.tsx (신규, ~80줄)

WebSocket `plan_progress` 이벤트를 수신하여 실시간 표시:

```tsx
export function PlanProgress() {
  const [plan, setPlan] = useState<PlanState | null>(null);

  useEffect(() => {
    // WebSocket 연결 — wss/status에서 plan_progress 수신
    const ws = new WebSocket(`ws://${location.host}/ws/status`);
    ws.onmessage = (e) => {
      const msg = JSON.parse(e.data);
      if (msg.type === "plan_progress") {
        setPlan(msg.data);
      }
    };
    return () => ws.close();
  }, []);

  if (!plan) return null;

  return (
    <div className="p-4 bg-card rounded-lg border">
      <div className="flex items-center justify-between mb-3">
        <h3 className="text-sm font-semibold">Execution Plan</h3>
        <span className="text-xs px-2 py-0.5 rounded bg-blue-500/20 text-blue-400">
          {plan.currentStep}/{plan.totalSteps}
        </span>
      </div>
      <p className="text-xs text-muted-foreground mb-3">{plan.goal}</p>
      {/* progress bar */}
      <div className="w-full h-1.5 bg-muted rounded-full mb-3">
        <div
          className="h-full bg-primary rounded-full transition-all"
          style={{ width: `${(plan.currentStep / plan.totalSteps) * 100}%` }}
        />
      </div>
      <p className="text-xs">
        Step {plan.currentStep}: {plan.stepDescription}
        <span className={cn("ml-2", statusColor(plan.stepStatus))}>
          [{plan.stepStatus}]
        </span>
      </p>
    </div>
  );
}
```
