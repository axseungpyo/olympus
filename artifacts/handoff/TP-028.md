## Objective

여러 에이전트(Brokkr, Heimdall)가 **동시에 실행**되며,
각 에이전트의 진행 상태를 **실시간으로 대시보드에 스트리밍**한다.

```
Odin(Brain)
  ├── Brokkr: TP-031 (백엔드) ██████░░ 60%
  ├── Heimdall: TP-032 (분석) ████████ 100% ✓
  └── Brokkr 대기: TP-033 (프론트, TP-032 의존)
```

---

## Context

### 현재 상태
- 에이전트는 한 번에 하나만 실행 가능 (StartAgentUseCase)
- PlannerUseCase는 step을 순차 실행만 함
- 에이전트 프로세스 이벤트(started/stopped)는 EventBus에 발행됨
- WebSocket으로 상태 변경 전달 인프라 존재

### 목표
1. PlannerUseCase에 **병렬 step 실행** 기능 추가
2. 에이전트 실행 시 **진행률 추정** (프로세스 출력 기반)
3. **agent_progress** 도메인 이벤트 → WebSocket → 대시보드
4. 대시보드에 **에이전트 진행 카드** 시각화

### 4. core/use-cases/agent/MonitorAgentUseCase.ts (신규, ~60줄)

에이전트 프로세스 출력을 모니터링하고 진행률을 추정:

```typescript
export class MonitorAgentUseCase {
  constructor(
    private readonly eventBus: IEventBus,
  ) {}

  startMonitoring(agent: string, tp: string, spawnResult: SpawnResult): void {
    let fileCount = 0;
    let lastFile = "";

    spawnResult.onStdout((line: string) => {
      // 파일 수정 감지 패턴 (codex/gemini 출력 파싱)
      const fileMatch = line.match(/(?:writing|modifying|creating)\s+(\S+)/i);
      if (fileMatch) {
        fileCount++;
        lastFile = fileMatch[1];
      }

      // 진행률 추정 (단순: 파일 수 기반, 최대 95%)
      const percent = Math.min(fileCount * 10, 95);

      this.eventBus.publish({
        type: AGENT_PROGRESS_EVENT,
        timestamp: Date.now(),
        payload: {
          agent,
          tp,
          percent,
          currentFile: lastFile,
          status: "running",
        },
      });
    });

    spawnResult.onExit((code) => {
      this.eventBus.publish({
        type: AGENT_PROGRESS_EVENT,
        timestamp: Date.now(),
        payload: {
          agent,
          tp,
          percent: 100,
          status: code === 0 ? "completed" : "failed",
        },
      });
    });
  }
}
```

### 6. PlannerUseCase 수정 — 병렬 실행 (~40줄)

```typescript
async executePlan(planId: string): Promise<ExecutionPlan> {
  // ... 기존 코드

  // step을 그룹으로 분류: 동시 실행 가능한 step은 Promise.all로 병렬 실행
  const groups = this.groupParallelSteps(plan.steps);

  for (const group of groups) {
    if (group.length === 1) {
      await this.executeStep(plan, group[0]);
    } else {
      // 병렬 실행
      await Promise.all(group.map(step => this.executeStep(plan, step)));
    }

    // 승인 대기 확인
    if (plan.status === "paused") return plan;
    // 실패 확인
    if (plan.status === "failed") return plan;
  }

  // ... 완료 처리
}

private groupParallelSteps(steps: PlanStep[]): PlanStep[][] {
  const groups: PlanStep[][] = [];
  let currentGroup: PlanStep[] = [];

  for (const step of steps) {
    if (step.parallel && currentGroup.length > 0) {
      currentGroup.push(step);
    } else {
      if (currentGroup.length > 0) groups.push(currentGroup);
      currentGroup = [step];
    }
  }
  if (currentGroup.length > 0) groups.push(currentGroup);

  return groups;
}
```

### 8. 대시보드 — agent-progress.tsx (신규, ~80줄)

```tsx
export function AgentProgress() {
  const [agents, setAgents] = useState<Map<string, AgentProgressData>>(new Map());

  useEffect(() => {
    const ws = new WebSocket(`ws://${location.host}/ws/status`);
    ws.onmessage = (e) => {
      const msg = JSON.parse(e.data);
      if (msg.type === "agent_progress") {
        setAgents(prev => {
          const next = new Map(prev);
          next.set(`${msg.data.agent}-${msg.data.tp}`, msg.data);
          // 완료된 에이전트는 5초 후 제거
          if (msg.data.status !== "running") {
            setTimeout(() => {
              setAgents(p => { const n = new Map(p); n.delete(`${msg.data.agent}-${msg.data.tp}`); return n; });
            }, 5000);
          }
          return next;
        });
      }
    };
    return () => ws.close();
  }, []);

  if (agents.size === 0) return null;

  return (
    <div className="space-y-2">
      {Array.from(agents.values()).map((agent) => (
        <div key={`${agent.agent}-${agent.tp}`} className="p-3 bg-card rounded-lg border">
          <div className="flex items-center justify-between mb-1">
            <span className="text-sm font-medium">
              {agent.agent === "brokkr" ? "Brokkr" : "Heimdall"} — {agent.tp}
            </span>
            <span className="text-xs">{agent.percent}%</span>
          </div>
          <div className="w-full h-1.5 bg-muted rounded-full">
            <div
              className="h-full rounded-full transition-all"
              style={{
                width: `${agent.percent}%`,
                backgroundColor: agent.status === "failed" ? "var(--destructive)" : "var(--primary)",
              }}
            />
          </div>
          {agent.currentFile && (
            <p className="text-xs text-muted-foreground mt-1 truncate">{agent.currentFile}</p>
          )}
        </div>
      ))}
    </div>
  );
}
```
