## Objective
Yggdrasil 대시보드에 Dark/Light 모드 토글을 추가한다. 현재 Night Sky(dark) 테마를 기본으로 유지하면서, 라이트 모드 전환을 지원한다. 토글 버튼은 Header 우측 설정 아이콘 옆에 배치한다.

## Architecture

### 구현 전략: CSS 변수 기반 테마 전환
- `globals.css`의 `@theme` 블록은 dark 기본값으로 유지
- `:root` / `html` 기본 = dark, `html.light` 클래스 추가 시 light 색상으로 오버라이드
- 컴포넌트에서 사용하는 hardcoded `text-slate-*` 클래스는 그대로 유지 (dark 모드에서 이미 잘 동작)
- light 모드에서는 CSS 변수(bg-primary, bg-secondary, border 등)만 오버라이드하고, slate-* 텍스트는 `html.light` 셀렉터로 일괄 반전

### 파일별 변경

#### 1. `src/yggdrasil/dashboard/app/globals.css`
- `html.light` 블록 추가:
  ```css
  html.light {
    --color-bg-primary: #f8fafc;      /* slate-50 */
    --color-bg-secondary: #f1f5f9;    /* slate-100 */
    --color-bg-tertiary: #e2e8f0;     /* slate-200 */
    --color-bg-elevated: #ffffff;
    --color-border: #cbd5e1;          /* slate-300 */
    --color-text-primary: #0f172a;    /* slate-900 */
    --color-text-secondary: #334155;  /* slate-700 */
    --color-text-muted: #64748b;      /* slate-500 */
    --color-text-faint: #94a3b8;      /* slate-400 */
    --color-link: #0891b2;            /* cyan-600 */
    --color-warn: #d97706;            /* amber-600 */
    --color-error: #dc2626;           /* red-600 */
  }
  ```
- `body` 스타일을 CSS 변수 참조로 변경:
  ```css
  body {
    background-color: var(--color-bg-primary);
    color: var(--color-text-secondary);
  }
  ```
- scrollbar 색상도 CSS 변수 참조
- `::selection` 색상도 light 모드 대응
- `html.light` 내 hardcoded slate 텍스트 반전 유틸리티:
  ```css
  html.light .text-slate-100 { color: #1e293b; }   /* slate-800 */
  html.light .text-slate-200 { color: #334155; }   /* slate-700 */
  html.light .text-slate-300 { color: #475569; }   /* slate-600 */
  html.light .text-slate-400 { color: #64748b; }   /* slate-500 */
  html.light .text-slate-500 { color: #94a3b8; }   /* slate-400 */
  ```
- `html.light` 에서 body bg와 scrollbar 재정의

#### 2. `src/yggdrasil/dashboard/lib/theme.ts` (신규)
- `ThemeMode` 타입: `"dark" | "light"`
- `getStoredTheme(): ThemeMode` — localStorage에서 `yggdrasil-theme` 키 조회, 기본값 `"dark"`
- `setStoredTheme(mode: ThemeMode): void` — localStorage 저장
- `applyTheme(mode: ThemeMode): void` — `document.documentElement`에 `dark`/`light` 클래스 토글
- `toggleTheme(): ThemeMode` — 현재 테마 반전 후 적용 + 저장, 새 테마 반환

#### 3. `src/yggdrasil/dashboard/components/ThemeToggle.tsx` (신규)
- 태양(light)/달(dark) 아이콘 토글 버튼
- 크기: 16x16 SVG, Header의 기존 설정 아이콘과 동일 스타일
- `text-slate-500 hover:text-slate-300 transition` 클래스
- 클릭 시 `toggleTheme()` 호출, 로컬 state로 현재 모드 관리
- `useEffect`로 마운트 시 `getStoredTheme()` → `applyTheme()` 호출

#### 4. `src/yggdrasil/dashboard/components/Header.tsx`
- `ThemeToggle` 임포트 및 설정 아이콘 왼쪽에 배치

#### 5. `src/yggdrasil/dashboard/app/layout.tsx`
- `<html>` 태그의 `className="dark"` 유지 (SSR 기본값)
- body에서 `text-slate-200` 제거 (CSS 변수가 처리)

#### 6. `src/yggdrasil/dashboard/app/page.tsx`
- Auth 모달의 `bg-[#0c0f1a]/80` → `bg-bg-primary/80` 변수 참조
- `bg-slate-100` 버튼 → light 모드에서도 눈에 띄도록 조건부 스타일 또는 반전 클래스

#### 7. `src/yggdrasil/dashboard/lib/constants.ts`
- `STATUS_CONFIG`, `TASK_STATUS_CONFIG`의 hex 색상은 유지 (차트/인라인 스타일이므로 테마 무관)

## Acceptance Criteria
1. Header 우측에 태양/달 아이콘 토글 버튼이 렌더링된다
2. 토글 클릭 시 dark ↔ light 전환이 즉시 반영된다
3. 선택한 테마가 localStorage에 저장되고, 새로고침 후에도 유지된다
4. Dark 모드: 현재 Night Sky 테마와 동일 (regression 없음)
5. Light 모드: 흰색/밝은 회색 배경, 어두운 텍스트, 모든 컴포넌트에서 가독성 확보
6. `npm test` 통과
7. `npm run build` 통과

## Out of Scope
- 시스템 테마(prefers-color-scheme) 자동 감지 (향후 추가 가능)
- 컴포넌트별 개별 테마 커스터마이징
