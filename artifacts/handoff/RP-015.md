# RP-015: Dark/Light 모드 토글

## Summary
Yggdrasil 대시보드에 Dark/Light 모드 토글을 구현했다. CSS 변수 기반으로 테마를 전환하며, `html.light` 클래스로 light 모드 색상을 오버라이드한다. Header 우측에 태양/달 아이콘 토글 버튼을 배치했고, localStorage로 선택을 유지한다.

## Context Digest
- Architecture: CSS 변수 기반 테마 전환. `@theme` 블록은 dark 기본값, `html.light` 셀렉터가 오버라이드.
- Key entry points: `lib/theme.ts` (로직), `ThemeToggle.tsx` (UI), `globals.css` (CSS 변수)
- Gotchas: hardcoded `text-slate-*` 클래스는 `html.light .text-slate-*` CSS 오버라이드로 반전됨 (!important 사용)
