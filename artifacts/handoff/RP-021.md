# RP-021: Use Case Layer + Controller Refactoring

## Summary
`routes -> controller -> use case -> repository` 흐름으로 서버 구조를 재편했다.  
Agent/Task/Odin 로직을 `core/use-cases/`로 이동하고, HTTP 엔드포인트는 `adapters/controllers/`에서 처리하도록 분리했다.
